Circle to Search

Visual search for your desktop

Getting Started

Press ⌘ + S to activate the search overlay

Once active, you can:

  • Draw a box to select text or areas
  • Release to highlight selected content
  • Press ESC to exit

Features

  • Full-screen overlay capture
  • Text detection and selection
  • Copy selected text to clipboard
  • Image-based search
